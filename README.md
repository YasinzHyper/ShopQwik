# ShopQwik

A full stack shopping application with an admin panel.

## Steps to Run:
### step 1: Server side
  cd server 
  
  npm install
  
  npm run dev
### step 2:
  - flutter pub get
  * open simulator (android) 
  - flutter run 

## Tech used: 

Server: Node.js, Express, Mongoose, MongoDB, Cloudinary.

Client: Flutter, Provider
